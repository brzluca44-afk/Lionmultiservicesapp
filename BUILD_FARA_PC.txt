Cum compilezi APK fara PC, direct din browser pe telefon:

1. Intra pe github.com si fa cont/login.
2. Creeaza un repository nou, de exemplu:
   LionMultiservicesApp

3. In repository apasa:
   Add file > Upload files

4. Incarca TOATE fisierele si folderele din acest proiect.
   Important: sa se vada in radacina:
   - app
   - build.gradle
   - settings.gradle
   - .github

5. Apasa Commit changes.

6. Intra sus pe tabul:
   Actions

7. Alege:
   Build APK

8. Apasa:
   Run workflow

9. Dupa ce termina, jos la Artifacts apare:
   Lion-Multiservices-APK

10. Descarci zip-ul, il dezarhivezi pe telefon si instalezi:
    app-debug.apk

Daca telefonul cere permisiune:
Setari > Securitate > Instalare aplicatii necunoscute > Permite pentru browser / Files.
